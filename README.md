# LUXORA 

LUXORA is a premium, high-performance cross-platform mobile application engineered using the **Flutter** framework. Designed specifically for cultural tourism and historical discovery, LUXORA leverages mobile computer vision, automated translation pipelines, and interactive AI to bridge the gap between ancient Egyptian history and modern visitors. 

The application interfaces seamlessly with a dedicated **Laravel REST API** hosted on secure Contabo cloud infrastructure.

---

##  Key Features

### 🔍 1. AI Scan & Recognition Hub
* **Mural Translation:** Processes ancient inscriptions and provides detailed, explanatory translations. Users can choose their target language (**Arabic** or **English**) and save the final translation output directly back to their device gallery.
* **Landmark Recognition:** Image classification models instantly identify key historical monuments, accurately detecting sites such as the **Karnak Temple**, **Luxor Temple**, **Hatshepsut Temple**, and more.

### 🤖 2. Interactive AI Historical Chatbot
* **Pocket Tour Guide:** An intelligent, conversational AI assistant that acts as an on-demand tour guide to answer deep historical queries about monuments, pharaohs, and ancient artifacts.
* **Bilingual Support:** Fully optimized to converse fluently in both **Arabic** and **English**, adapting instantly to the user's interface language.

### 🗺️ 3. Explore & Interactive Mapping
* **Monument Discovery:** An interactive catalog where users can browse, learn about, and explore various ancient monuments.
* **Geographical Mapping:** Each monument profile includes integrated map locations to help users navigate to or locate sites easily.
* **Favorites List:** Users can bookmark specific historical locations and save them to a personalized favorites collection for quick reference.

### 🌐 4. Global Bilingual Support
* The entire application features full localization support, allowing users to switch the entire interface smoothly between **Arabic** and **English**.

## 🛠️ Tech Stack & Ecosystem

* **Frontend Framework:** Flutter & Dart (Stable Channel)
* **Backend Infrastructure:** Laravel REST API (Deployed on Contabo Cloud Infrastructure)
* **Performance & Profiling:** Flutter DevTools (CPU Profiler, Memory Monitor) & Apache JMeter
